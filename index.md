<h3> <a name="1225">12月24日 是时候展示项目成果了 </a></h3>
  <h4>1.项目的想法：</h4>
    <small>
    <p>实现一个可以通过手动控制的机械臂，然后应用该机械臂做出各种动作，比如夹起一个小球，放在指定的位置<br><b>改进方案：</b>通过蓝牙远程控制，再添加一个记忆装置，实现自动化。</p>
  </small>
  <h4>2.制作过程：</h4>
    <p><small><b>a.需要使用的材料：</b><br/>舵机（个数根据自己设计的机械臂自由度而定）、电源（anduino需要5V的电源，舵机的电源根据舵机而定）、旋转电位计（这里买的是10K欧，300度转角）、Aduino UNO板、面包板、电线若干、亚克力板材和3D打印耗材。<br/>
       <b>b.需要使用的仪器和软件：</b><br/>3D打印机、激光切割机、Fushion 360、Auto CAD。<br/>
       <b>第一步：</b>用Fushion 360建立机械臂的躯干的3D模型，需要自己测量舵机的尺寸，设计机械臂的造型和尺寸。建好模型后使用3D打印机打印。下面是一些建模的图片：<br/>
         <img src="images/jixiebi.png" width="50%" height="50%" align="left"/>

         <img src="images/机械.png" width="50%" height="50%" align="right"/>
      <br>

       <b>第二步：</b>做一个大小差不多的简易的控制机械臂，同样需要先设计好控制机械臂的造型和尺寸，先用Auto Cad绘制切割的图形，再使用激光切割机切割，然后用旋转电位计作为关节连接好控制机械臂。<br/>
       <b>第三步：</b>Aduino UNO板上程序的编译，下面展示我们的机械臂的代码：<br/>
       <pre>
         #include<Servo.h>
         Servo myservo1,myservo2,myservo3,myservo4,myservo5;
         int potpin1 = A0;
         int potpin2 = A1;
         int potpin3 = A2;
         int potpin4 = A3;
         int potpin5 = A4;
         int val1,val2,val3,val4,val5;
         void setup()
         {
           // Serial.begin(9600);        //必要时启动串口来进行调试
           myservo1.attach(3);
           myservo2.attach(6);
           myservo3.attach(9);
           myservo4.attach(11);
           myservo5.attach(12);
         }
         void loop()
         {
           //根据你的情况修改所有带感叹号注释的数，获得准确的模仿效果
           val1 = analogRead(potpin1);
           val1 = map(val1, 100/*！*/, 788/*！*/, 0, 180);
           val2 = analogRead(potpin2);
           val2 = map(val2, 180/*！*/, 888/*！*/, 0, 180);
           val3 = analogRead(potpin3);
           val3 = map(val3, 0/*！*/, 898/*！*/, 0, 180);
           val4 = analogRead(potpin4);
           val4 = map(val4, 180/*！*/, 883/*！*/, 0, 180);
           val5 = map(val5, 180/*！*/, 883/*！*/, 0, 180);

           // Serial.print("val1=");
           // Serial.print(analogRead(0));
           // Serial.print(" val2=");
           // Serial.print(analogRead(1));
           // Serial.print(" val3=");
           // Serial.print(analogRead(2));
           // Serial.print(" val4=");
           // Serial.println(analogRead(3));

           myservo1.write(val1);
           myservo2.write(val2);
           myservo3.write(val3);
           myservo4.write(val4);
           myservo5.write(val5);
           delay(10);             //修改括号里的数字，改变机械臂的运动速度。
         }
       </pre>
       <b>PS:</b>如果出现舵机发抖的现象，可以使用滤波算法，可以使舵机运行的更加流畅。另外，如果UNO板和机械臂是分开供电的话，必须要保证它们的地线连在一起，否则舵机也会抖动得很厉害。<br/>
       <b>第四步：</b>用激光切割机做一个底板，固定好两个机械臂和面包板，连接线路。最后调试机械臂。
    </small></p>
